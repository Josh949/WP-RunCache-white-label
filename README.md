# WP-WL-Backend-Plugin
A Wordpress MU plugin that replaces WordPress branding with your own. It also hides the admin side Theme file editor and plugin editor.



Directions:
1. Add your own svg file to /white-label/images/WL_logo.svg
2. Replace its name to WL_logo.svg  
3. Move these files into wp-content/mu-plugins
4. Thats it, enjoy!
